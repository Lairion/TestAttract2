ИНСТРУКЦИЯ ПО УСТАНОВКЕ ПРОЕКТА

ВНИМАНИЕ!

Перед тем как ставить проект, убедитесть что на вашем устройстве есть следующее програмное обеспечение:
- Python 3
- pip
- virtualenv
- git

Устанавливаем проект.

1. Создаем виртуальное окружение(ВО) в удобном для вас месте. Далее откройте в том месте терминал (или cmd для Windows plt.). Введите следующую команду.

	virtualenv -p python3 <любое имя для вашего виртуального окружения>

2. Перейдите во внутрь ВО, и активируйте его.

сd < имя ВО>

На unix:

source bin/activate

На Windows:

scripts/activate

Все дальнейшие действия выполнять в активированном ВО

3. Клонируйте репозиторий или скачайте zip по этому адрессу: https://github.com/Lairion/TestAttract2

Клонируем:

git clone https://github.com/Lairion/TestAttract2.git

4. Далее переходим в папку с проектом:

	cd TestAttract2/

Устанавливаем необходимое ПО для запуска проекта.

	pip install -r req.txt

5. Делаем миграцию для базы.

python manage.py makemigrations
python manage.py migrate

6. Создаем superuser.
Вводим следуюшую команду:

python manage.py createsuperuser

7. Запускаем проект.

 	python manage.py runserver

Настраиваем проект.

1. Импортируем пользователей.

Импорт пользователей можно сделать в Admin Panel. Импорт можно произвести в Shop>User> Кномка Import. Нужно выбрать файл в формате csv и указать тип файла.
Пример шаблона таблицы для импорта находится в папке import_examp/examp.csv

2. Добавляем продукты в базу.
Продукты можно занести в базу через Admin Panel. У каждого продукта должна быть Категория. Сначало создайте категории а потом добавьте для них шаблоны продуктов.
После этого когда пользователь будет создавать заказ, он будет создавать продукт используя шаблон продукта. Все это можно создать в Разделе Product>category> Create, Product>Product_template Create. Также шаблоны продукта можно создавать на прямую из категорий.

3. Использовать приложение можно начиная с http://<ip>:<port>/list_customer.
При создфния меню пользователя стрался придерживатся концепта шаблону задания.
